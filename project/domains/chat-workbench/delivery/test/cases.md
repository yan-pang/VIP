## 冒烟用例

- TC-101 扫码登录成功 → TC-110 全号聚合 → TC-112 指派 → TC-118 发送 → TC-120 送达判定。

- 接待闭环:TC-106 接收 → TC-112 指派 → TC-118 发送 → TC-120 判定 → TC-117 结束。
- 主动发起闭环:TC-125 / TC-126。
